Rust Cal

rust-cal Link to heading

This is a project I built as part of my attempt to begin learning Rust. I thought it’d be fun to easily and quickly display my university timetable for the day in my terminal (as I’m using my laptop throughout most of the day). It accesses my timetable by making a request to the iCalendar URL provided by the university for timetable syncing, and uses the ical crate to parse the resulting .ics file

It was a good introduction to using Cargo and crates and I have been adding new features that I find useful.
